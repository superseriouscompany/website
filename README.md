# Super Serious Website

## Requirements

1. rvm
1. ruby 2.1.2
1. rails 4.2.0
1. http://postgresapp.com/

## Installation

    $ bundle install

## Running

    $ rails s

## Managing database

    $ rake db:schema:load db:seed
